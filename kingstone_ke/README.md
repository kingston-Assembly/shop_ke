# Kingstone KE

A modern, animated Kenyan e‑commerce single‑page website for a store located at Kingstone Plaza, Nairobi West, Nairobi, Kenya. Features a product catalog, cart, and simulated M‑Pesa checkout.

## Quick start

- Open `index.html` in a browser, or run a static server:

```bash
# from the project root
python3 -m http.server 8080
# then open http://localhost:8080/
```

## Features

- Animated hero and scroll‑reveal sections
- Product search and sorting
- Cart with persistent storage (localStorage)
- Checkout page with simulated M‑Pesa payment and Kenyan phone validation
- Responsive, mobile‑friendly UI

## Customization

- Update products in `assets/js/data.js`
- Adjust styles in `assets/css/styles.css`
- Branding icons live in `assets/icons/`
